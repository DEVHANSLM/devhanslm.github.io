Desenvolupament web és una disciplina que engloba la creació i el manteniment de llocs web. Inclou diverses tasques, com la programació del costat del client i del servidor, així com el disseny i la gestió de bases de dades. HTML és el llenguatge de marques utilitzat per estructurar el contingut, mentre que CSS o Tailwind permeten estilitzar-lo per donar-li una aparença visual. JavaScript afegeix interactivitat i dinàmica als llocs web. Això permet crear experiències d'usuari més completes i atractives. Les tecnologies del backend, com ara Node.js o PHP, són essencials per gestionar les dades i la lògica del servidor. La formació contínua en programació de les noves tecnologies és imprescindible per mantenir-se actualitzat en aquest camp. El treball en equip i l'adaptabilitat són claus per l'eficiència en la creació de projectes web cada vegada més complexos i innovadors, i les eines com frameworks com: React, Angular, Vue.js, Flask, Django, Ruby, Laravel, Kotlin, Next.js, Bootstrap. i llibreries faciliten molt aquest procés.
